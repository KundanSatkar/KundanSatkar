## Featured Projects

### 🤖 [Synapse Street — AI Multi-Agent Stock Analysis](https://github.com/KundanSatkar)
> Built in 24 hours at a hackathon. LangGraph-based multi-agent system analyzing 3,000+ U.S. stocks with Qdrant vector search and Hadoop HDFS for storage.  
> **18% return on backtested portfolios · 60% reduction in manual analysis time**

### 🏥 [Osteoporosis Risk Prediction](https://github.com/KundanSatkar)
> Clinical ML system using XGBoost, Logistic Regression, and SHAP explanations on a 10K-patient dataset. Designed for interpretability — non-technical healthcare staff can act on outputs.  
> **AUC: 0.87 · False-negative rate reduced by 12%**

### 📚 [Amazon Review Sentiment (Hadoop + ML)](https://github.com/KundanSatkar)
> Distributed NLP pipeline processing 8M+ review records using MapReduce and Spark. Ensemble models (Random Forest, Gradient Boosting, SVM) with 5-fold cross-validation.  
> **84% classification accuracy · 15% variance reduction**

### 🗃️ [Online Music Store — SQL Analytics System](https://github.com/KundanSatkar)
> Normalized relational schema + 15+ complex SQL queries for KPI reporting on 500K+ transactions. Optimized with multi-column indexing.  
> **35% faster query execution · 2x faster stakeholder delivery**
